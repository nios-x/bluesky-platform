## 🚀 Quick Start (7 minutes)

```bash
# 1. Get Claude API key
# https://console.anthropic.com → Create API key

# 2. Add to .env.local
ANTHROPIC_API_KEY=sk-ant-xxxxx

# 3. Update app/layout.tsx or app/page.tsx
import { AIChatbotRefactored } from '@/components/ai-chatbot-refactored'
<AIChatbotRefactored />

# 4. Start
npm run dev

# 5. Test
# Open http://localhost:3000
# Click chat bubble → Start chatting!
```
